# 🗓️ 캘린더 웹 사이트 제작 프로젝트

## 프로젝트 소개 

임대차 계약을 하기 앞서 배울 수 있고 계약에 도움을 주는 서비스

- 퀴즈를 통한 임대차 계약 배우기
- 계약을 할때 체크리스트를 통해 도움 받기
- 전세 사기 위험도 확인하기

<br/>
